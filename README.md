🛒 Market Basket Optimization Dashboard

Understanding customer purchasing behavior is one of the most valuable parts of retail analytics.

This project focuses on identifying product associations, buying patterns, and cross-selling opportunities using transaction-level data.

Built an end-to-end Market Basket Analysis Dashboard using Power BI, SQL, and Python to uncover how products interact inside customer baskets and how those relationships can help businesses improve sales strategies.

Instead of only analyzing individual product sales, this project focuses on:

Product relationships
Basket combinations
Customer buying behavior
Cross-selling opportunities
Product association patterns
🚀 Project Overview

The dashboard helps answer important business questions such as:

Which products are frequently purchased together?
Which combinations increase basket size?
Which items act as connector products?
What buying patterns can improve merchandising decisions?
Which products support stronger recommendation systems?
